Como faço pra colocar imagens?

  • Primeiro baixamos a imagem de desejamos colocar no nosso projeto;
  • Vamos utilizar a TAG IMG para que seja possível adicionar a imagem;
  • Em SRC vamos dar o caminho pra encontrar essa imagem, após iremos colocar o nome dessa imagem.
